# 学习路径工作流

## 工作流概述

```
设定目标 → 规划路径 → 执行学习 → 整理笔记 → 知识内化 → 输出分享
```

## 流程步骤

### 步骤 1：设定学习目标
**使用 Skill**：`goal-setter`

明确学习目标：
- 要学什么
- 达到什么水平
- 什么时间完成

**触发方式**：
```
我想学习 [技能/知识]，请帮我设定学习目标
```

**输出**：
- SMART 目标描述
- 分阶段里程碑
- 成功标准定义

### 步骤 2：规划学习路径
**使用 Skill**：`learning-tracker`

制定学习计划：
- 学习内容分解
- 资源整理
- 时间安排

**触发方式**：
```
请帮我制定学习 [技能] 的详细计划
```

**输出**：
- 分阶段学习计划
- 推荐学习资源
- 每周学习安排

### 步骤 3：执行学习
**使用 Skill**：`concept-explainer`

学习过程中：
- 理解新概念
- 解决疑问
- 深化理解

**触发方式**：
```
请解释一下 [概念] 是什么？
```

**输出**：
- 概念的清晰解释
- 简单易懂的类比
- 实际应用示例

### 步骤 4：整理笔记
**使用 Skill**：`note-taker`

学习后整理：
- 提取关键点
- 结构化笔记
- 建立联系

**触发方式**：
```
请帮我整理这段学习内容的笔记
```

**输出**：
- 结构化笔记
- 关键概念总结
- 复习要点

### 步骤 5：知识内化
**使用 Skill**：`knowledge-base`

构建知识体系：
- 整合新旧知识
- 建立知识地图
- 定期复习

**触发方式**：
```
请帮我将 [主题] 的学习内容整合到知识库
```

**输出**：
- 知识卡片
- 知识关联图
- 复习计划

### 步骤 6：输出分享
**使用 Skill**：`tutorial-creator`

通过教授巩固：
- 写教程/博客
- 制作分享材料
- 费曼学习法实践

**触发方式**：
```
请帮我将 [主题] 的学习成果整理成一篇教程
```

**输出**：
- 完整教程文档
- 循序渐进的结构
- 练习和检验点

## 快速启动

### 启动学习
```
我想学习 [技能/知识]：
- 当前水平：[描述]
- 目标水平：[描述]
- 可用时间：每周 X 小时
请帮我规划学习路径
```

### 学习中求助
```
我在学习 [主题] 时遇到了问题：
[问题描述]
请帮我解释
```

### 学习总结
```
我完成了 [阶段/课程] 的学习，请帮我：
1. 整理学习笔记
2. 总结关键概念
3. 制定复习计划
```

### 输出巩固
```
我想把 [主题] 的学习成果写成教程分享：
- 目标读者：[描述]
- 重点内容：[列表]
```

## 学习效率技巧

### 番茄工作法
```
25分钟专注学习 → 5分钟休息 → 重复
每4个番茄 → 长休息15-30分钟
```

### 间隔重复
```
Day 1: 学习
Day 2: 第一次复习
Day 4: 第二次复习
Day 7: 第三次复习
Day 14: 第四次复习
```

### 费曼学习法
```
1. 学习概念
2. 教授给别人（或假装教授）
3. 发现理解不清的地方
4. 回去深入学习
5. 用更简单的语言重新解释
```
