# 项目启动工作流

## 工作流概述

```
项目定义 → 目标设定 → 方案讨论 → 决策确认 → 计划制定 → 启动会议
```

## 流程步骤

### 步骤 1：项目定义
**使用 Skill**：`goal-setter`

明确项目范围和目标：
- 项目背景和目的
- 预期成果
- 成功标准

**触发方式**：
```
我要启动一个 [项目名称] 项目，请帮我定义项目目标
```

**输出**：
- 项目目标陈述
- 可衡量的指标
- 边界定义

### 步骤 2：方案头脑风暴
**使用 Skill**：`brainstormer`

探索实现方案：
- 发散思考可能的方案
- 探索不同路径
- 收集创意

**触发方式**：
```
针对 [项目目标]，请帮我头脑风暴可能的实现方案
```

**输出**：
- 方案清单
- 各方案的初步评估
- 创新思路

### 步骤 3：方案决策
**使用 Skill**：`decision-maker`

选择最佳方案：
- 评估各方案优缺点
- 风险分析
- 做出决策

**触发方式**：
```
请帮我分析以下方案，选择最佳实现路径：
- 方案A: [描述]
- 方案B: [描述]
```

**输出**：
- 方案对比分析
- 推荐方案及理由
- 风险预案

### 步骤 4：计划制定
**使用 Skill**：`task-planner`

制定详细计划：
- 工作分解结构
- 时间安排
- 资源规划

**触发方式**：
```
请帮我制定 [项目] 的详细执行计划
```

**输出**：
- 任务分解清单
- 里程碑定义
- 时间线规划

### 步骤 5：启动演示准备
**使用 Skill**：`presentation-maker`

准备启动会议材料：
- 项目介绍 PPT
- 关键信息展示
- 团队动员

**触发方式**：
```
请帮我准备项目启动会议的演示材料
```

**输出**：
- PPT 大纲和内容
- 演讲要点
- 视觉设计建议

### 步骤 6：启动会议
**使用 Skill**：`meeting-notes`

召开启动会议并记录：
- 会议议程
- 讨论记录
- 行动项跟踪

**触发方式**：
```
请帮我记录项目启动会议的内容
```

**输出**：
- 会议纪要
- 决议事项
- 后续行动项

## 快速启动

### 完整项目启动
```
我要启动一个项目：
- 项目名称：[名称]
- 背景：[描述]
- 预期成果：[描述]
请帮我完成项目启动的准备工作
```

### 快速项目规划
```
请帮我快速规划 [项目]：
1. 定义项目目标
2. 分解主要任务
3. 制定初步时间线
```

### 启动会议准备
```
项目计划已确定，请帮我准备启动会议：
1. 设计会议议程
2. 准备演示材料
3. 制作会议纪要模板
```

## 项目启动文档模板

```markdown
# [项目名称] 项目启动文档

## 1. 项目概述
- 项目背景
- 项目目的
- 项目范围

## 2. 目标与成功标准
- 主要目标
- 可衡量指标
- 验收标准

## 3. 关键干系人
- 项目发起人
- 项目经理
- 核心团队成员
- 其他相关方

## 4. 实施方案
- 选定方案概述
- 技术路线
- 关键假设

## 5. 项目计划
- 主要里程碑
- 阶段划分
- 时间安排

## 6. 资源需求
- 人力资源
- 预算
- 工具和设备

## 7. 风险管理
- 主要风险
- 应对措施

## 8. 沟通计划
- 会议安排
- 汇报机制
- 协作工具

## 9. 附录
- 相关文档
- 参考资料
```
