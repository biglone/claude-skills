# 每周规划工作流

## 工作流概述

```
周回顾 → 目标检视 → 任务规划 → 习惯追踪 → 决策处理
```

## 最佳执行时间

建议在每周日或周一早上进行，预留 30-60 分钟。

## 流程步骤

### 步骤 1：周回顾
**使用 Skill**：`weekly-review`

回顾过去一周：
- 完成了什么
- 遇到了什么问题
- 学到了什么

**触发方式**：
```
请帮我做本周回顾，我这周完成了：
1. [事项1]
2. [事项2]
...
```

**输出**：
- 周报/周回顾文档
- 本周亮点总结
- 待改进事项

### 步骤 2：目标检视
**使用 Skill**：`goal-setter`

检查目标进度：
- 年度目标进展
- 季度 OKR 状态
- 月度目标完成度

**触发方式**：
```
请帮我检视目标进度，我的目标是：
- [目标1]
- [目标2]
```

**输出**：
- 目标进度报告
- 差距分析
- 调整建议

### 步骤 3：任务规划
**使用 Skill**：`task-planner`

规划下周任务：
- 根据目标确定重点
- 分解为具体任务
- 安排时间和优先级

**触发方式**：
```
请帮我规划下周的任务，重点是：
1. [重点1]
2. [重点2]
```

**输出**：
- 周任务清单
- 每日安排建议
- 优先级排序

### 步骤 4：习惯追踪
**使用 Skill**：`habit-tracker`

追踪习惯完成情况：
- 本周习惯打卡回顾
- 分析完成率
- 调整习惯策略

**触发方式**：
```
请帮我分析本周的习惯完成情况：
- 运动：完成5/7天
- 阅读：完成7/7天
...
```

**输出**：
- 习惯完成率报告
- 趋势分析
- 改进建议

### 步骤 5：决策处理
**使用 Skill**：`decision-maker`

处理待决策事项：
- 梳理需要决策的问题
- 分析各选项
- 做出决定

**触发方式**：
```
我有个决定需要做：[描述问题]
选项有：
- A: [选项A]
- B: [选项B]
```

**输出**：
- 决策分析
- 建议选项
- 行动计划

## 快速启动

### 完整周规划
```
请帮我完成本周的周回顾和下周规划：
1. 本周完成的事项：[列表]
2. 下周的重点目标：[列表]
3. 我的习惯追踪数据：[数据]
```

### 简版周回顾
```
请帮我快速回顾本周：
- 完成了什么
- 学到了什么
- 下周重点是什么
```

### 每日检查
```
今天是周X，请帮我检查：
- 今日任务完成情况
- 明日安排
- 需要调整的事项
```

## 每周规划检查清单

### 周回顾
- [ ] 回顾本周目标完成情况
- [ ] 总结本周亮点和收获
- [ ] 分析未完成事项的原因
- [ ] 记录经验教训

### 习惯检查
- [ ] 更新习惯打卡记录
- [ ] 分析习惯完成率
- [ ] 调整习惯策略

### 下周规划
- [ ] 确定下周 3 个重点
- [ ] 分解为具体任务
- [ ] 安排到日历
- [ ] 预留缓冲时间

### 目标对齐
- [ ] 检视月度/季度目标
- [ ] 确保周计划与目标一致
- [ ] 调整必要的目标

## 时间安排建议

```
周日晚间 / 周一早晨

09:00 - 09:15  周回顾
09:15 - 09:25  目标检视
09:25 - 09:40  任务规划
09:40 - 09:50  习惯追踪
09:50 - 10:00  决策处理
```
