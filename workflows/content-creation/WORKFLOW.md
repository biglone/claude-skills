# 内容创作工作流

## 工作流概述

```
选题构思 → 大纲设计 → 内容撰写 → 优化润色 → 翻译适配 → 发布
```

## 流程步骤

### 步骤 1：选题构思
**使用 Skill**：`brainstormer`

产生内容创意：
- 头脑风暴
- 热点结合
- 读者需求分析

**触发方式**：
```
请帮我头脑风暴关于 [主题] 的内容选题
```

**输出**：
- 选题列表
- 各选题的角度和亮点
- 推荐优先级

### 步骤 2：大纲设计
**使用 Skill**：`outline-creator`

设计内容结构：
- 文章大纲
- 章节安排
- 要点规划

**触发方式**：
```
请为 [选题] 设计一个详细的大纲
```

**输出**：
- 多级大纲
- 各章节要点
- 预计篇幅

### 步骤 3：内容撰写
**使用 Skill**：`blog-writer` 或 `technical-writer`

撰写正文内容：
- 博客文章使用 `blog-writer`
- 技术文档使用 `technical-writer`

**触发方式**：
```
请根据大纲撰写文章内容
```

**输出**：
- 完整的文章内容
- 配图建议
- SEO 优化建议

### 步骤 4：优化润色
**使用 Skill**：`blog-writer`

优化文章质量：
- 语言润色
- 逻辑调整
- 标题优化

**触发方式**：
```
请帮我优化这篇文章的表达
```

### 步骤 5：多语言适配
**使用 Skill**：`translator`

翻译和本地化：
- 中英互译
- 保持风格
- 文化适配

**触发方式**：
```
请将这篇文章翻译成 [目标语言]
```

### 步骤 6：发布准备
*（手动完成）*

- 格式调整
- 配图设计
- 平台发布

## 快速启动

### 完整创作流程
```
我想写一篇关于 [主题] 的文章，请帮我：
1. 产生几个创意选题
2. 设计文章大纲
3. 撰写初稿
```

### 快速优化
```
请帮我优化这篇文章：
1. 改进标题
2. 润色语言
3. 增强可读性
```

### 多语言版本
```
请将这篇中文文章翻译成英文，保持技术准确性
```

## 内容创作检查清单

### 选题阶段
- [ ] 选题有价值
- [ ] 目标读者明确
- [ ] 差异化角度

### 撰写阶段
- [ ] 大纲逻辑清晰
- [ ] 内容有深度
- [ ] 举例充分
- [ ] 语言流畅

### 发布阶段
- [ ] 标题吸引人
- [ ] 格式美观
- [ ] 配图合适
- [ ] SEO 优化
- [ ] 多平台适配

## 内容类型模板

### 技术博客
```
选题 → 大纲 → technical-writer → 优化 → 发布
```

### 观点文章
```
选题 → brainstormer → blog-writer → 润色 → 发布
```

### 教程类
```
选题 → outline-creator → technical-writer → 代码审查 → 发布
```
