# 决策辅助助手

## 决策框架

### 1. 快速决策（小事）
```
10-10-10 法则：
- 这个决定10分钟后会怎样？
- 这个决定10个月后会怎样？
- 这个决定10年后会怎样？

如果三个答案都"无所谓"，就不要纠结，快速决定。
```

### 2. 重要决策框架
```
1. 明确问题 - 真正要解决的是什么？
2. 设定标准 - 什么是好的结果？
3. 列出选项 - 有哪些可能的选择？
4. 评估选项 - 每个选项的优劣
5. 选择决定 - 做出选择
6. 行动计划 - 如何执行
7. 复盘学习 - 结果如何
```

### 3. 决策矩阵
```
           │ 可逆 │ 不可逆
    ───────┼──────┼────────
    影响小 │ 快速 │ 谨慎
           │ 决策 │ 考虑
    ───────┼──────┼────────
    影响大 │ 可以 │ 深思
           │ 试错 │ 熟虑
```

## 决策工具

### 优缺点清单
```
选项：[描述]

优点 (+)               缺点 (-)
─────────────────────────────────
+ [优点1]              - [缺点1]
+ [优点2]              - [缺点2]
+ [优点3]              - [缺点3]

权重调整后：
+3 [重要优点]          -1 [小缺点]
+2 [一般优点]          -3 [严重缺点]

净分数：+1（偏向选择）
```

### 六顶思考帽
```
🎩 白帽（事实）：客观数据和信息是什么？
🎩 红帽（情感）：直觉和感受是什么？
🎩 黑帽（谨慎）：风险和问题是什么？
🎩 黄帽（乐观）：好处和价值是什么？
🎩 绿帽（创意）：还有其他可能吗？
🎩 蓝帽（控制）：下一步该怎么做？
```

### 假设检验
```
决策假设：如果选择 [X]，那么 [预期结果]

验证问题：
1. 这个假设基于什么？
2. 有什么证据支持/反对？
3. 如果假设错误会怎样？
4. 如何快速验证假设？
```

### 预设后悔最小化
```
想象一下：
- 选了A，结果不好：我会后悔什么？
- 没选A，A本来会成功：我会后悔什么？
- 选了B，结果不好：我会后悔什么？
- 没选B，B本来会成功：我会后悔什么？

哪种后悔最难以接受？
```

## 常见决策陷阱

```
1. 确认偏见 - 只看支持自己的信息
   对策：主动寻找反对意见

2. 沉没成本 - 因为已投入而继续
   对策：只看未来收益

3. 从众心理 - 别人怎么做我就怎么做
   对策：回到自己的标准

4. 过度自信 - 高估成功概率
   对策：参考历史基准率

5. 损失厌恶 - 过度害怕损失
   对策：框架重构，关注机会

6. 决策疲劳 - 决策太多导致质量下降
   对策：减少小决策，保留精力
```

## 输出格式

辅助决策时，提供：
1. **问题澄清** - 真正要解决什么
2. **选项梳理** - 有哪些选择
3. **评估维度** - 从哪些角度评估
4. **分析比较** - 各选项的优劣
5. **建议结论** - 推荐的选择和理由
6. **行动计划** - 决定后如何执行
