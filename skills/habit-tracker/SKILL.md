# 习惯追踪助手

## 习惯养成科学

### 习惯循环
```
        ┌─────────────┐
        │   提示      │
        │   (Cue)     │
        └──────┬──────┘
               │
               ▼
        ┌─────────────┐
        │   渴望      │
        │  (Craving)  │
        └──────┬──────┘
               │
               ▼
        ┌─────────────┐
        │   反应      │
        │ (Response)  │
        └──────┬──────┘
               │
               ▼
        ┌─────────────┐
        │   奖励      │
        │  (Reward)   │
        └─────────────┘
```

### 习惯养成四定律

```
建立好习惯：
1. 让它显而易见（Cue）    - 设计环境提示
2. 让它有吸引力（Craving） - 与喜欢的事绑定
3. 让它简单易行（Response）- 从2分钟开始
4. 让它令人满足（Reward）  - 即时奖励自己

戒除坏习惯：
1. 让它隐而不见 - 移除触发因素
2. 让它没吸引力 - 重新定义意义
3. 让它难以执行 - 增加阻力
4. 让它令人不满 - 建立惩罚机制
```

## 习惯模板

### 习惯设计卡
```markdown
# 习惯：[习惯名称]

## 基本信息
**类型**：🟢 养成 / 🔴 戒除
**关联目标**：[[目标]]
**开始日期**：YYYY-MM-DD
**目标天数**：30/60/90 天

## 习惯设计

### 1. 提示设计（何时何地）
- **时间**：[具体时间]
- **地点**：[具体地点]
- **触发行为**：在 [已有习惯] 之后
- **环境设计**：[如何让提示更明显]

### 2. 渴望设计（动机绑定）
- **内在动机**：为什么想养成这个习惯
- **习惯叠加**：做完 [习惯] 后，我可以 [喜欢的事]
- **身份认同**：我是一个 [什么样的人]

### 3. 反应设计（具体行动）
- **2分钟版本**：[最简单的开始方式]
- **标准版本**：[正常的执行方式]
- **完整版本**：[理想的执行方式]

### 4. 奖励设计（正向反馈）
- **即时奖励**：完成后 [奖励自己什么]
- **追踪方式**：[如何记录]
- **里程碑奖励**：
  - 7天：[奖励]
  - 30天：[奖励]
  - 90天：[奖励]

## 障碍预判
| 可能的障碍 | 应对策略 |
|------------|----------|
| [障碍1] | [策略] |
| [障碍2] | [策略] |

## 承诺宣言
"我承诺每天 [时间] 在 [地点] 进行 [习惯]。"

签名：__________ 日期：__________
```

## 习惯养成阶段

```
第1阶段：起步期（1-7天）
- 重点：不断链，降低标准也要完成
- 策略：2分钟规则，从最小版本开始

第2阶段：适应期（8-21天）
- 重点：建立稳定的执行节奏
- 策略：固定时间地点，减少决策

第3阶段：巩固期（22-66天）
- 重点：应对特殊情况
- 策略：制定 if-then 计划

第4阶段：自动化（66天+）
- 重点：不需要意志力
- 策略：可以适当增加难度
```

## 输出格式

追踪习惯时，提供：
1. **习惯设计** - 完整的四要素设计
2. **追踪表格** - 可视化的打卡记录
3. **数据分析** - 完成率和趋势
4. **反馈建议** - 基于数据的改进建议
5. **激励信息** - 连续天数、里程碑
