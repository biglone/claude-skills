# 周回顾助手

## 周回顾流程

### GTD 周回顾清单
```
1. 收集 (Collect)
   - [ ] 清空所有收件箱（邮件、消息、笔记）
   - [ ] 收集零散的想法和待办
   - [ ] 整理物理桌面和电子桌面

2. 处理 (Process)
   - [ ] 处理所有新收集的内容
   - [ ] 决定下一步行动或归档

3. 回顾 (Review)
   - [ ] 回顾日历（过去和未来）
   - [ ] 回顾项目清单
   - [ ] 回顾等待清单
   - [ ] 回顾"可能/也许"清单

4. 规划 (Plan)
   - [ ] 确定下周重点
   - [ ] 安排具体任务
   - [ ] 预留缓冲时间
```

## 周回顾问题

### 反思问题
```
回顾过去：
1. 这周最重要的成就是什么？
2. 这周最大的挑战是什么？如何应对的？
3. 有什么是应该做但没做的？为什么？
4. 有什么是做了但不应该花那么多时间的？
5. 这周学到了什么？

规划未来：
1. 下周最重要的3件事是什么？
2. 有什么是可以提前准备的？
3. 有什么潜在的障碍？如何预防？
4. 这周的经验如何应用到下周？
```

## 输出格式

生成周报/周回顾时，提供：
1. **完成情况** - 清晰的任务状态
2. **成果亮点** - 关键成就
3. **问题分析** - 遇到的挑战
4. **下周计划** - 明确的目标
5. **反思总结** - 经验教训
