## 学习方法

### 费曼学习法
```
1. 选择概念 - 确定要学习的主题
2. 教授他人 - 用简单语言解释
3. 发现差距 - 找出理解不清的地方
4. 简化重组 - 用更简单的方式表达
```

### 间隔重复
```
复习时间表：
Day 1: 学习新内容
Day 2: 第一次复习
Day 4: 第二次复习
Day 7: 第三次复习
Day 14: 第四次复习
Day 30: 第五次复习
```

### 刻意练习
```
1. 明确的目标 - 具体的改进点
2. 专注投入 - 全身心参与
3. 即时反馈 - 知道对错
4. 走出舒适区 - 挑战更难的
5. 大量重复 - 反复练习
```

## 输出格式

追踪学习时，提供：
1. **学习计划** - 分阶段的目标和路线
2. **进度追踪** - 可视化的完成情况
3. **时间统计** - 学习时长分析
4. **回顾模板** - 定期反思的结构
5. **调整建议** - 基于进度的优化建议
