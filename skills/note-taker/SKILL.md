# 笔记整理助手

## 笔记方法

### 1. 康奈尔笔记法
```
┌─────────────────────────────────────┐
│           标题/日期                  │
├──────────┬──────────────────────────┤
│          │                          │
│  关键词   │       主要笔记           │
│  问题    │       - 要点1            │
│  提示    │       - 要点2            │
│          │       - 细节...          │
│  (课后)  │       (课中)             │
│          │                          │
├──────────┴──────────────────────────┤
│              总结                    │
│         (课后24小时内)               │
└─────────────────────────────────────┘
```

### 2. 思维导图法
```
              ┌── 分支1.1
        ┌─ 分支1 ─┼── 分支1.2
        │        └── 分支1.3
中心主题 ─┼─ 分支2 ─── 分支2.1
        │        ┌── 分支3.1
        └─ 分支3 ─┴── 分支3.2
```

### 3. 要点列表法
```markdown
# 主题

## 核心概念
- **概念1**: 定义和解释
- **概念2**: 定义和解释

## 关键要点
1. 要点一
2. 要点二
3. 要点三

## 示例/案例
- 案例1: 说明
- 案例2: 说明

## 疑问/待深入
- [ ] 问题1
- [ ] 问题2
```

### 4. 卡片盒笔记法 (Zettelkasten)
```markdown
---
id: 202401150001
title: 笔记标题
tags: [标签1, 标签2]
links: [[相关笔记1]], [[相关笔记2]]
---

# 笔记内容

[用自己的话写下理解]

## 来源
[原始出处]

## 关联
- 与 [[笔记A]] 的关系：...
- 与 [[笔记B]] 的关系：...
```

## 笔记整理技巧

### 提取要点
```
原则：
1. 找出主题句（通常在段首/段尾）
2. 识别关键词和专业术语
3. 注意信号词（首先、其次、关键是、总之）
4. 区分事实和观点
5. 用自己的话重新表述
```

### 建立联系
```
方法：
- 与已有知识的联系
- 与其他笔记的联系
- 与实际应用的联系
- 与个人经历的联系
```

### 定期回顾
```
间隔重复时间表：
- 1天后：第一次复习
- 3天后：第二次复习
- 7天后：第三次复习
- 14天后：第四次复习
- 30天后：第五次复习
```

## 输出格式

整理笔记时，输出包含：
1. **核心要点** - 3-5个关键点
2. **结构化内容** - 清晰的层级
3. **个人理解** - 用自己的话解释
4. **关联链接** - 相关知识点
5. **行动项** - 可执行的下一步
