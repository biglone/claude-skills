# 教程创建助手

## 教程类型

### 1. 快速入门 (Quickstart)
- 目标：5-15 分钟完成
- 内容：最基本的使用方法
- 受众：完全新手

### 2. 完整教程 (Tutorial)
- 目标：30-60 分钟完成
- 内容：系统学习某个主题
- 受众：想深入学习的人

### 3. 操作指南 (How-to Guide)
- 目标：解决特定问题
- 内容：具体步骤
- 受众：有基础的使用者

### 4. 课程系列 (Course)
- 目标：掌握一个领域
- 内容：多个模块/章节
- 受众：系统学习者

## 教学设计原则

### 1. 明确学习目标
```markdown
学完本教程，你将能够：
- [ ] 理解 [概念]
- [ ] 使用 [工具/方法]
- [ ] 完成 [具体任务]
```

### 2. 循序渐进
```
基础 → 进阶 → 高级
理论 → 实践 → 应用
简单 → 复杂 → 综合
```

### 3. 动手实践
```
每个知识点 = 讲解 + 示例 + 练习
理论占比：30%
实践占比：70%
```

### 4. 及时反馈
```
每个步骤后验证结果
提供常见错误的解决方案
设置检查点确认理解
```

## 写作技巧

### 步骤说明
```markdown
# 好的步骤说明

### 步骤 1：创建项目目录

首先，我们需要创建一个新的项目目录：

```bash
mkdir my-project
cd my-project
```

运行后，你会进入新创建的目录。

> 💡 **提示**：如果你使用 Windows，可以用 `md` 替代 `mkdir`

---

# 避免的写法

1. 创建目录
2. 进入目录
3. 初始化

（太简略，没有具体命令和验证）
```

### 代码示例
```markdown
# 好的代码示例

```python
# 导入必要的库
import requests

# 发送 GET 请求
response = requests.get('https://api.example.com/users')

# 检查响应状态
if response.status_code == 200:
    users = response.json()
    print(f"获取到 {len(users)} 个用户")
else:
    print(f"请求失败: {response.status_code}")
```

**代码说明**：
- 第 2 行：导入 requests 库用于发送 HTTP 请求
- 第 5 行：发送 GET 请求获取用户列表
- 第 8-12 行：根据状态码处理响应
```

## 教程质量检查

### 检查清单
- [ ] 学习目标明确
- [ ] 前提条件清晰
- [ ] 步骤完整可执行
- [ ] 每步有验证方法
- [ ] 代码已测试通过
- [ ] 包含常见问题解答
- [ ] 提供延伸学习资源
- [ ] 难度循序渐进

## 输出格式

创建教程时，根据类型输出：
1. **学习目标** - 明确告诉读者会学到什么
2. **前提条件** - 需要准备什么
3. **步骤内容** - 详细的操作步骤
4. **检查点** - 如何验证每步成功
5. **练习/作业** - 巩固所学
6. **延伸资源** - 进一步学习
