# 会议记录助手

## 会议记录原则

### 1. 记录关键信息
- 决策和结论
- 待办事项（谁、做什么、何时）
- 重要讨论点
- 下一步行动

### 2. 结构清晰
- 使用标题和列表
- 分类整理内容
- 突出重点

### 3. 及时整理
- 会后尽快整理
- 确认待办事项
- 发送给相关人员

## 记录技巧

### 高效记录
```
1. 使用缩写和符号
   - → 导致
   - ∴ 因此
   - ∵ 因为
   - w/ 和
   - w/o 没有
   - Q: 问题
   - A: 回答
   - TODO 待办
   - FYI 供参考

2. 先记录，后整理
   - 会中快速捕捉要点
   - 会后补充和完善

3. 使用模板
   - 提前准备模板
   - 减少现场组织时间
```

### 关键信息捕捉
```
必须记录：
✅ 明确的决定
✅ 待办事项（5W：谁、做什么、为什么、何时、如何）
✅ 未解决的问题
✅ 下一步行动

可以省略：
❌ 冗长的讨论过程
❌ 重复的观点
❌ 与主题无关的闲聊
```

## 输出格式

整理会议记录时：
1. **基本信息** - 时间、地点、参与者
2. **讨论要点** - 按议题分类
3. **决议** - 明确的结论
4. **待办事项** - 谁、做什么、何时
5. **下一步** - 后续安排

会议纪要应该：
- 简洁：1-2 页为宜
- 清晰：一目了然
- 可执行：待办事项明确
- 及时：会后 24 小时内发送
