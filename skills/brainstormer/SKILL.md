# 头脑风暴助手

## 头脑风暴原则

### 基本规则
```
1. 数量优先 - 先求多，再求好
2. 延迟判断 - 不在产生阶段批评
3. 欢迎疯狂 - 越离奇的想法可能越有价值
4. 借力打力 - 在别人想法上叠加
```

### 思维发散流程
```
阶段1：准备（5分钟）
- 明确问题
- 收集背景信息
- 进入开放心态

阶段2：发散（15-20分钟）
- 自由联想
- 记录所有想法
- 不做评判

阶段3：收敛（10分钟）
- 分类整理
- 初步筛选
- 组合优化
```

## 创意方法

### SCAMPER 法
```
S - Substitute（替代）
    用什么可以替代？
    换个材料/人/流程会怎样？

C - Combine（组合）
    能和什么结合？
    不同想法如何融合？

A - Adapt（适应）
    能从别处借鉴什么？
    其他行业怎么做的？

M - Modify（修改）
    能放大/缩小什么？
    能改变形状/颜色/顺序吗？

P - Put to other uses（其他用途）
    还能用在哪里？
    有其他场景吗？

E - Eliminate（消除）
    能去掉什么？
    简化后会怎样？

R - Reverse/Rearrange（逆向/重排）
    能颠倒吗？
    反过来会怎样？
```

### 六顶思考帽
```
🎩 白帽（事实）
问：有哪些客观数据和事实？
    缺少什么信息？

🎩 红帽（情感）
问：直觉告诉你什么？
    你对此有什么感觉？

🎩 黑帽（谨慎）
问：有什么风险和问题？
    为什么可能失败？

🎩 黄帽（乐观）
问：有什么好处和价值？
    为什么可能成功？

🎩 绿帽（创意）
问：还有什么可能？
    能否换个角度？

🎩 蓝帽（控制）
问：下一步该怎么做？
    如何组织思考？
```

### 逆向思维
```
正向问题：如何让用户更满意？
逆向问题：如何让用户更不满意？
           (然后避免这些)

正向问题：如何提高效率？
逆向问题：如何降低效率？
           (然后消除这些障碍)

步骤：
1. 明确原始问题
2. 反转为相反问题
3. 列出相反问题的答案
4. 再次反转，获得原问题的方案
```

### 随机输入法
```
方法：
1. 随机选择一个词/图片/物品
2. 列出它的特征和联想
3. 将这些联想与问题连接

示例：
问题：如何让团队会议更高效？
随机词：咖啡机

咖啡机联想 → 应用到会议
- 按需供应 → 只在需要时开会
- 自助服务 → 让大家自己决定参与
- 快速出品 → 设定时间限制
- 不同口味 → 不同形式的会议
```

### 强制关联法
```
步骤：
1. 列出两个看似无关的概念
2. 强制寻找它们的关联
3. 从关联中产生新想法

示例：
概念A：教育
概念B：游戏

强制关联：
- 游戏化学习
- 关卡式课程
- 成就系统激励
- 多人协作任务
- 失败可重来
```

## 创意激发问题

### 重新定义问题
```
- 如果这不是问题，而是机会呢？
- 真正的问题是什么？
- 问题背后的问题是什么？
- 谁会不认为这是问题？
- 10年后这还是问题吗？
```

### 改变视角
```
- 用户会怎么看？
- 竞争对手会怎么做？
- 孩子会怎么解决？
- 专家会怎么处理？
- 外行会怎么想？
- 另一个行业会怎么做？
```

### 打破假设
```
- 我们假设了什么？
- 如果假设是错的呢？
- 有哪些"不可能"其实可能？
- 规则可以打破吗？
- 如果资源无限会怎样？
- 如果时间紧迫10倍呢？
```

### 极端思考
```
- 做到极致会怎样？
- 完全不做会怎样？
- 规模扩大100倍呢？
- 缩小到只有1个用户呢？
- 免费会怎样？
- 收费10倍呢？
```

## 创意评估矩阵

```
           │  影响力高
           │
    快速    │    重点
    胜利    │    突破
           │
───────────┼───────────
           │
    备选    │    长期
    方案    │    投资
           │
           │  影响力低
    实现简单 ←───────→ 实现困难
```

## 输出格式

头脑风暴时，提供：
1. **问题重构** - 多角度理解问题
2. **想法列表** - 大量发散的想法
3. **分类整理** - 按主题或可行性分组
4. **亮点挑选** - 最有潜力的几个想法
5. **下一步建议** - 如何进一步发展
