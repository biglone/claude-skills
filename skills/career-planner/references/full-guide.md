## 职业规划框架

### 自我认知三环模型
```
        ┌─────────────────┐
        │   热爱的事      │
        │  (Passion)      │
        └────────┬────────┘
                 │
    ┌────────────┼────────────┐
    │            │            │
    ▼            ▼            ▼
┌───────┐    ┌───────┐    ┌───────┐
│擅长的 │ ∩  │热爱的 │ ∩  │市场   │
│事     │    │事     │    │需要的 │
│(Good) │    │(Love) │    │(Paid) │

    │            │            │
    └────────────┼────────────┘
                 │

           理想职业
```

### 职业发展阶段
```
阶段1：探索期（0-3年）
- 尝试不同方向
- 建立基础技能
- 找到兴趣所在

阶段2：建立期（3-7年）
- 深耕专业领域
- 建立个人品牌
- 扩展人脉网络

阶段3：成长期（7-15年）
- 成为领域专家
- 承担更大责任
- 影响力扩大

阶段4：成熟期（15年+）
- 战略性思考
- 培养下一代
- 留下职业遗产
```

## 职业转型指南

### 转型评估
```
适合转型的信号：
✓ 持续对当前工作无激情
✓ 长期没有成长感
✓ 对新领域有强烈兴趣
✓ 愿意为转型付出代价

不适合转型的信号：
✗ 只是逃避当前问题
✗ 对新领域了解肤浅
✗ 不愿意从头开始
✗ 只看到光鲜的一面
```

### 转型策略
```
渐进式转型：
1. 在现有工作中寻找交叉机会
2. 利用业余时间学习和实践
3. 通过副业验证兴趣和能力
4. 逐步过渡

跨越式转型：
1. 储备足够的经济缓冲
2. 全职投入学习和转型
3. 接受初期的薪资下降
4. 快速积累新领域经验
```

### 可迁移技能
```
硬技能迁移：
- 数据分析 → 任何需要数据驱动的岗位
- 项目管理 → 任何需要协调资源的岗位
- 编程能力 → 多种技术和非技术岗位

软技能迁移：
- 沟通表达 → 销售、运营、产品等
- 问题解决 → 咨询、创业、管理等
- 学习能力 → 任何新领域
```

## 输出格式

职业规划时，提供：
1. **自我评估** - 价值观、技能、兴趣分析
2. **方向建议** - 基于分析的职业方向
3. **发展路径** - 具体的阶段性目标
4. **行动计划** - 可执行的下一步
5. **风险提示** - 需要注意的挑战
